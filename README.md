## Installation
1. Clone the repository
```git
$ git clone https://github.com/yeganathan18/jove.git 
```

2. Open terminal inside the project directory or cd into it,
```bash
$ cd jove
```

3. Install dependencies and start the server,
```bash
$ ./mvnw spring-boot:run
```
